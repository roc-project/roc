Supported platforms
*******************

.. seealso::

   * :doc:`/development/continuous_integration`

What is supported
-----------------

The following platforms are officially supported.

Operating systems:

* GNU/Linux
* macOS
* Android

Compilers:

* GCC (>= 4.4)
* Clang (>= 3.4)

What is tested
--------------

The following specific platform configurations are tested on a regular basis.

Linux distributions:

* Ubuntu (20.04, 18.04, 16.04, 14.04)
* Debian (stable)
* Fedora (latest)
* CentOS (latest)
* openSUSE (Leap)
* Arch Linux (latest)
* Alpine Linux (latest)

Android versions:

* ABI: arm64-v8a (ARM 64-bit), armeabi-v7a (ARM 32-bit), x86_64 (Intel 64-bit), x86 (Intel 32-bit)
* API: 29 (Android 10), 24 (Android 7)
* NDK: 21

macOS versions:

* macOS 10.15 (Catalina)

Hardware architectures:

* x86_64
* i686
* ARMv6
* ARMv7 (Cortex-A 32-bit)
* ARMv8 (Cortex-A 64-bit)
